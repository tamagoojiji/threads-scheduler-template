データ削除手順（Data Deletion Instructions）

最終更新: 2026年4月24日

本ページは、スレッズスケジューラー（以下「本サービス」）利用者が自己データを削除するための手順を案内します。

本サービスの構造上の特徴
本サービスは「利用者所有モデル」を採用しており、投稿データ・認証情報・スプレッドシートのすべてが利用者自身のアカウント配下に保存されています。当方（tamago）は投稿内容・認証情報に一切アクセスできません。そのため、データ削除も利用者自身が各サービスで実施していただく形になります。

1. 利用者自身が削除するデータ

(A) スプレッドシートの削除（Apps Scriptと認証情報も同時削除）

  1. Googleドライブを開く
  2. 「スレッズスケジューラー」のスプシを選択
  3. 右クリック → 「ゴミ箱へ移動」
  4. ゴミ箱を空にする

→ スプシに紐付いたApps Scriptプロジェクトも同時に削除されます。Apps Script Properties に保存されていた Threadsアクセストークン・Webhook URL等もこのタイミングで消去されます。

(B) 時間トリガーの削除（スプシ削除前に推奨）

  1. スプシメニュー「スレッズスケジューラー → 🗑 トリガーを削除」を実行

→ スプシを削除する前にトリガーを停止しておくと、削除過渡期に余計な実行が走るのを防げます。

(C) Meta Developerアプリの削除

  1. developers.facebook.com にログイン
  2. 「マイアプリ」から該当アプリを開く
  3. 「設定」→「一般」→「アプリの削除」

→ Threadsアクセストークンも無効化されます。

(D) Discordチャンネル / Webhookの削除（オプション）

  1. Discordで通知用に作ったチャンネルを開く
  2. チャンネル設定 → 連携サービス → ウェブフック → 削除
  3. 必要であればチャンネル自体も削除

2. 当方（tamago）保有データの削除請求

当方が保有するデータは以下2点のみです:

  • 連絡用メールアドレス
  • アカウント名（ハンドル）

削除請求方法

以下の内容で商品ページ記載のメールアドレス宛にご連絡ください:

件名: 【データ削除請求】スレッズスケジューラー

本文:
・登録アカウント名: （ハンドル名）
・登録メールアドレス: （メール）
・削除希望: 全データ削除をお願いします

ご連絡受領後、7営業日以内に削除処理を完了し、完了通知メールをお送りします。

3. Stripe決済データの削除

決済情報はStripe社が保持しており、当方は参照権限がありません。決済情報の削除はStripe社に直接ご請求ください。
