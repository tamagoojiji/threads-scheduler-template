利用規約

制定日: 2026年4月24日 / 最終更新: 2026年4月24日

本規約は、スレッズスケジューラー（以下「本サービス」）の導入代行サービスおよび関連サポートに関する利用条件を定めるものです。

第1条（サービス概要）

本サービスは、Threads自動予約投稿の仕組みを利用者環境にセットアップする「導入代行サービス」です。セットアップ完了後の運用は利用者自身の責任で行うものとします。

第2条（利用料金）

  1. 導入代行費: Zoomセットアップ同伴（目安1〜1.5時間）込みで、申込時に1回払い
  2. 月額サポートプラン（任意）: エラー対応・仕様変更通知等の継続サポート
  3. 決済はStripe社を通じて行います

第3条（セットアップ後の責任範囲）

重要: セットアップ完了時点で、以下は利用者の所有物・管理責任となります。
  • 利用者のGitHubリポジトリ
  • 利用者のMeta Developerアプリ
  • 利用者のGoogleスプレッドシート
  • Threadsアクセストークン等の認証情報

第4条（配信保証レベル）

本サービスは at-least-once 配信 + 事後整合 を提供します:

  • 少なくとも1回は投稿が試行されます
  • Threads API側の仕様上、ごくまれに重複投稿が発生する可能性があります
  • GitHub Actionsのcron実行遅延により、指定時刻から±5〜15分の投稿ズレが発生することがあります
  • 投稿成功/失敗は利用者のDiscordに通知されます

第5条（禁止事項）

利用者は以下の行為を行ってはなりません:

  • スパム投稿、なりすまし、違法コンテンツの投稿
  • Meta Platform Terms、Threads利用規約への違反
  • 第三者への本サービス（テンプレート・手順書等）の転売・譲渡・公開
  • 当方のシステム・他利用者への妨害行為

第6条（免責事項）

  1. 投稿内容の適法性・倫理性は利用者の責任です
  2. Threads API / Google Sheets API / GitHub Actions 等の仕様変更・障害・停止による影響について、当方は責任を負いません
  3. 利用者の設定ミス・認証情報漏洩・運用ミスによる損害について、当方は責任を負いません
  4. 本サービスは「現状有姿」で提供され、特定目的への適合性は保証しません
  5. 利用者の投稿がMeta側で削除・アカウント停止された場合、当方は一切責任を負いません

第7条（サポート範囲）

7.1 月額サポートプラン契約者

  • エラー発生時の調査・助言
  • Meta/Google仕様変更時の更新通知
  • 対応時間・範囲は事前合意の範囲内

7.2 非契約者

  • 個別サポートは提供しません
  • セットアップ時に提供した手順書・FAQを参照して自己解決してください

第8条（当方の停止権）

利用者が本規約・Meta Platform Terms・法令に違反した場合、当方は事前通知なく以下を実施する権利を有します:

  • サポート提供の即時停止
  • 月額サポートプランの解約

第9条（解約フロー）

  1. 月額サポートプラン契約者は、書面（メール）により解約を申し出できます
  2. 当月末日で解約処理、翌月分の課金は停止します
  3. 利用者所有のシステム（GitHubリポジトリ・スプシ・Meta Developerアプリ）は解約後もそのまま稼働継続します
  4. 当方は解約から30日以内に、保有する連絡先情報を削除します

第10条（準拠法・管轄）

本規約は日本法を準拠法とし、本サービスに関する紛争は当方所在地を管轄する裁判所を第一審の専属的合意管轄とします。
